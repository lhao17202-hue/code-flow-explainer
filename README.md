# Code Flow Explainer Skill

A Claude skill for explaining code by tracing runtime execution, data flow, inputs, outputs, intermediate values, side effects, return values, and downstream usage.

This skill is designed for users who want to understand not just what code is for, but exactly what happens when it runs.

## Skill

See [SKILL.md](SKILL.md).

## Use cases

Use this skill for prompts such as:

- "Walk through this function step by step."
- "Trace the data flow of this line."
- "What does this function return?"
- "How is this variable produced?"
- "Simulate this code running with example values."
- "按执行顺序讲解这段代码。"
- "从数据流讲这个函数。"
